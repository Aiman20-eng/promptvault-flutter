# PromptVault

تطبيق لإدارة وحفظ البرومبتات الخاصة بالذكاء الاصطناعي تم تطويره باستخدام Flutter و Provider.

## المميزات

* عرض البرومبتات حسب التصنيفات
* البحث والفلاتر
* إضافة للمفضلة
* حفظ البرومبتات في مجموعات
* نسخ البرومبتات بسرعة
* واجهة عربية متجاوبة
* دعم الوضع الداكن
* إدارة حالة باستخدام Provider

## التقنيات المستخدمة

* Flutter
* Dart
* Provider
* ChangeNotifier
